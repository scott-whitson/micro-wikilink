# Wikilink Plugin

Navigate between markdown files using [[wikilinks]].

## Commands

- `wikilink.follow` (Alt-g): Follow the [[link]] under the cursor
- `wikilink.back` (Alt-b): Go back to the previous file
- `wikilink.open` (Alt-o): Fuzzy-find a note in the vault via fzf

## Settings

- `wikilink.vault`: Absolute path to your vault directory (default: current working directory)
